### 修复 Python 3.9+ 类型提示语法兼容性

用户 Linux 服务器 Python 版本 ≤ 3.8，`list[float]`、`str | None` 等 PEP 585/604 语法报 SyntaxError。

改动文件（`~/.config/opencode/skills/skill-creator/` 下）：
- **scripts/aggregate_benchmark.py**：`list[float]` → `List[float]`，`dict[str, list]` → `Dict[str, List]`
- **scripts/generate_report.py**：`list[dict]` → `List[dict]`，`tuple[int, int]` → `Tuple[int, int]`
- **scripts/improve_description.py**：`str | None` → `Optional[str]`，`list[dict]` → `List[dict]`
- **scripts/run_eval.py**：`list[dict]` → `List[dict]`，`dict[str, list[bool]]` → `Dict[str, List[bool]]`
- **scripts/run_loop.py**：`tuple[list[dict], list[dict]]` → `Tuple[List[dict], List[dict]]`，`str | None` → `Optional[str]`
- **scripts/utils.py**：`tuple[str, str, str]` → `Tuple[str, str, str]`，`list[str]` → `List[str]`
- **eval-viewer/generate_review.py**：同上模式，约 15 处替换
